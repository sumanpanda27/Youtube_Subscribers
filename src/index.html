Welcome to the Youtube Subscribers fetch homepage.

The API provided by this application contains the URL as following below:

  • BASE URL:Home in URL : http://localhost:3000
  • GET METHOD:/subscribers returns an array of subscribers present in the database.
  • GET METHOD:/subscribers/names returns an array of subscribers with only two fields (name & subscribedChannel).
  • GET METHOD:/subscribers/:id URL Provides the details of a subscriber with the given ID.(Default id is 652a76cda4400381906578ce)

    • You can also type "ID" manually in URL to get the details of stored id details
